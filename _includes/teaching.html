Object detection and classification with deep learning

Project on Github


I trained a convolutional neural network to detect and classify images of six-sided dice in real time. I used the TensorFlow Object Detection framework to construct, train and deploy a multi-object detection model, and augmented the trained model for use on mobile devices using TFLite. To take advantage of cloud computing resources, I also implemented a docker-based deployment to train the model on AWS GPU EC2 instances and the Google Cloud Services AI Platform.
